# 明末·烽火（FPS_MingMo）

> 一款基于网页的明末题材第一人称对战游戏（类似 CS 的爆破模式），三阵营对决：**明军 / 农民军 / 清军**，每营各有优势。火铳 + 冷兵器混合战斗，三段装填、烟雾遮挡、刀兵冲脸一应俱全。

![tech](https://img.shields.io/badge/Three.js-r160-blue) ![tech](https://img.shields.io/badge/Zero-Build-ff69b4) ![tech](https://img.shields.io/badge/License-MIT-green)

在线试玩：**https://showibin.github.io/FPS_MingMo/**（GitHub Pages，自动部署 main 分支）

---

## 特性

- **三阵营差异化**
  - 明军：火器精良，装填快、精度高，可部署拒马/铳架
  - 农民军：速度+近战，移动最快、近战加成、人数优势
  - 清军：重甲护体，血厚、护甲高、擅长弓箭远程稳定
- **核心机制：火铳**
  - 威力大、装填慢、开火有烟雾遮挡、精度不稳定
  - 近距离一枪致命，远距命中率骤降
  - 装填时被刀兵冲脸高风险
- **近战系统**
  - 火铳打一发后切刀冲锋
  - 轻击 / 重击 / 格挡
  - 残血敌人可处决（带风险）
- **地图：山寨关隘**
  - A 点：火药库 ｜ B 点：粮仓
  - 中路：村道 ｜ 边路：屋顶/小巷 ｜ 狙击位：箭楼
  - 高低差克制、窄巷发挥刀兵
- **历史见证**
  - 战斗中浮现《明史》《纪效新书》《武备志》原句
  - 关键人物、战役时间线穿插
- **A/B 爆破回合制**
  - 进攻方携带火药，安放至 A 或 B
  - 防守方拆除或全歼进攻方获胜
  - 多回合计分
- **炫酷 UI**
  - 水墨主色 + 火焰红蓝边
  - 血墨晕染、屏角烟雾、命中震屏
  - 战场小地图、阵营旌旗
  - 动态准星扩散、击杀鸣金、战鼓回合、主菜单余烬上升
- **手机端适配**
  - 左摇杆移动 + 右半屏滑动视角
  - 开火/瞄准/跳/蹲/冲刺/装填/换械/重击/处决触屏按钮
  - 窄屏 HUD 自适应缩放
- **零构建**：纯静态资源，开箱即玩。Three.js 走 CDN。

## 本地运行

任意静态服务器即可：

```powershell
# 任选一种
npx http-server -p 8080
# 或
python -m http.server 8080
```

然后浏览器打开 `http://localhost:8080`。**必须 HTTPS 或 localhost 才能 PointerLock**。

## 操作

| 键 | 动作 |
|---|---|
| W/A/S/D | 移动 |
| 鼠标 | 视角 |
| 鼠标左键 | 开火 / 近战轻击 / 拉弓 |
| 鼠标右键 | 瞄准（铳）/ 格挡（刀） |
| E | 近战重击；在 A/B 点安放或拆解火药 |
| R | 装填 |
| 1 | 切火铳 |
| 2 | 切冷兵器 |
| 3 | 清军切角弓 |
| F | 处决残血敌人 |
| Shift | 冲刺 |
| Space | 跳 |
| C | 蹲 |
| M | 静音 |
| Esc | 解锁鼠标 |

手机端：左摇杆移动，右半屏滑动转视角，触屏按钮开火/瞄准/跳/蹲/冲刺/装填/换械/重击（安放）/处决。

## 设计取舍

依用户意见，第一张图不做城池大地图，而做**山寨关隘**；确立火铳为差异化核心，再叠加近战补刀，最后加阵营特性与历史见证。详见 `docs/历史资料.md`。
