# 部署说明

本游戏为纯静态网站（无构建），只用 CDN 加载 Three.js / Howler，可一键部署到任意静态主机。

## 推荐：Vercel / Netlify / Cloudflare Pages

### Vercel
1. `npm i -g vercel`
2. 在 `FPS_MingMo/` 目录下执行 `vercel`
3. 框架选项选择 **Other**，构建命令留空，输出目录 `.`（或 `/`）
4. 部署完成后获得 `https://xxx.vercel.app`

### Netlify
1. 拖拽整个 `FPS_MingMo/` 文件夹到 https://app.netlify.com/drop
2. 或使用 CLI：`netlify deploy --prod --dir=.`

### Cloudflare Pages
1. Pages → Create a project → Direct Upload
2. 上传 `FPS_MingMo/` 全部内容
3. Build command 留空

### GitHub Pages
1. 推送到 GitHub 仓库后
2. Settings → Pages → Source: main / root
3. 等待几分钟，访问 `https://<user>.github.io/<repo>/`

> 注意：务必部署到 HTTPS 域名，否则浏览器不支持 PointerLock 与某些音频权限。

## 本地静态服务器

```powershell
npx http-server -p 8080
# 或
python -m http.server 8080
```
访问 `http://localhost:8080`。

## 环境要求

- 现代浏览器（Chrome/Edge 110+ 推荐）
- 支持 WebGL2
- PointerLock（鼠标锁定）
- 建议独立显卡，否则场景简化
